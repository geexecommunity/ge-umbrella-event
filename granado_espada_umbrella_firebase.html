1. ไปที่ console.firebase.google.com → Add project
2. ตั้งชื่อ project → กด Continue จนเสร็จ
3. ไปที่ Realtime Database → Create database

5. Copy ค่าจาก firebaseConfig มากรอกในช่องด้านบน
